# Decisiones Técnicas

## Decisión 1
Se comenzará con HTML, CSS y JavaScript antes de usar React.

### Motivo
Permite comprender mejor la lógica base, el DOM, los eventos y la persistencia antes de introducir abstracciones.

---

## Decisión 2
La primera persistencia será con localStorage.

### Motivo
Es simple, rápida de implementar y suficiente para un MVP.

---

## Decisión 3
Más adelante se migrará a IndexedDB.

### Motivo
Ofrece mejor estructura y escalabilidad para almacenamiento local.

---

## Decisión 4
La sincronización entre dispositivos se resolverá con Supabase.

### Motivo
Permite autenticación, base de datos y acceso desde distintos dispositivos usando una solución moderna y gratuita en etapa inicial.

---

## Decisión 5
El proyecto se desarrollará por fases y no intentando construir todo desde el inicio.

### Motivo
Reduce frustración, permite validar la base del proyecto y mejora el aprendizaje progresivo.
