# Roadmap del Proyecto

## Fase 0 — Planeación
Objetivo: definir qué se va a construir y cómo.

### Tareas
- Definir visión del proyecto
- Definir MVP
- Crear documentación base
- Definir estructura inicial

---

## Fase 1 — MVP con HTML, CSS y JavaScript
Objetivo: construir la primera versión funcional.

### Funcionalidades
- Registrar ingreso
- Registrar gasto
- Mostrar lista de transacciones
- Eliminar transacción
- Calcular balance total
- Guardar información en localStorage

### Aprendizajes
- DOM
- eventos
- formularios
- arrays y objetos
- renderizado dinámico
- persistencia local

---

## Fase 2 — Mejorar persistencia
Objetivo: pasar de localStorage a IndexedDB.

### Funcionalidades
- Persistencia más robusta
- Mejor manejo de grandes cantidades de datos
- Base local más estructurada

### Aprendizajes
- IndexedDB
- asincronía
- estructura de almacenamiento local

---

## Fase 3 — Migración a React
Objetivo: reconstruir la app usando componentes.

### Funcionalidades
- Componentización
- Separación de responsabilidades
- Estado reactivo
- Hooks básicos

### Aprendizajes
- useState
- useEffect
- props
- componentes reutilizables
- arquitectura frontend moderna

---

## Fase 4 — Integración con Supabase
Objetivo: sincronizar datos entre dispositivos.

### Funcionalidades
- Registro de usuario
- Inicio de sesión
- Guardado en la nube
- Consulta de movimientos desde cualquier dispositivo

### Aprendizajes
- autenticación
- base de datos relacional
- operaciones CRUD
- integración frontend-backend

---

## Fase 5 — Escalado del producto
Objetivo: volver la app más completa y presentable.

### Posibles mejoras
- Dashboard
- gráficas
- metas de ahorro
- categorías personalizadas
- gastos recurrentes
- exportación de reportes
- modo oscuro
- diseño responsive avanzado
